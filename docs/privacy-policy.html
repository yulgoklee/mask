📋 기본 원칙

마스크 알림 앱은 서버가 없습니다.
모든 데이터는 사용자의 기기 내에만 저장되며, 외부 서버로 개인정보를 전송하거나 수집하지 않습니다.

본 방침은 앱 내에서 사용되는 권한과 데이터의 처리 방식을 안내합니다.

시행일: 2026년 4월 1일

📍 위치 권한

항목 내용
수집 목적 사용자 주변 가장 가까운 미세먼지 측정소 자동 선택
수집 시점 사용자가 '현재 위치로 자동 감지' 버튼을 누를 때만
저장 방식 위치 좌표는 저장하지 않음. 선택된 측정소명만 기기 내 저장
외부 전송 없음 전송 안 함

위치 권한을 허용하지 않아도 앱 사용이 가능합니다. 이 경우 측정소를 직접 검색하여 설정하실 수 있습니다.

🔔 알림 권한

외출 전 미세먼지 알림, 귀가 알림, 실시간 경보 등을 위해 알림 권한을 요청합니다.

  • 알림은 사용자가 설정한 시간과 조건에 따라서만 발송됩니다.
  • 알림 내용은 기기 내에서 생성되며 외부 서버를 거치지 않습니다.
  • 알림 권한은 언제든지 기기 설정에서 해제할 수 있습니다.

🌍 외부 서비스 연동

① 한국환경공단 에어코리아 API

  • 미세먼지 실시간 측정값 및 예보 조회에 사용됩니다.
  • API 요청 시 측정소 이름만 전달되며, 개인정보는 포함되지 않습니다.
  • 에어코리아 개인정보처리방침: www.airkorea.or.kr

② Google AdMob (광고)

  • 앱 내 정보 화면에 배너 광고가 표시될 수 있습니다.
  • AdMob은 광고 최적화를 위해 기기 식별자를 사용할 수 있습니다.
  • Google 개인정보처리방침: policies.google.com/privacy
  • 광고 개인화 설정: 기기 설정 → Google → 광고에서 변경 가능합니다.
